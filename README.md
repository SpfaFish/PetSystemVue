# petsystem
一款真正人性化的宠物管理系统前端，包含贴吧交流，私信，关注，宠物系统等一系列各种功能，使用 Vue + Element-ui 实现。
## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
